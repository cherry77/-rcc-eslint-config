# 引入包(三选一)

```
npm install @rcc-pkg/eslint-config -D

yarn add @rcc-pkg/eslint-config --dev

pnpm add @rcc-pkg/eslint-config -D
```
# 项目中使用
## 安装(三选一)
```
npm install eslint -D

yarn add eslint --dev

pnpm add eslint -D
```

在项目根目录下创建 .eslintrc.json 文件
```json
{
  "root": true,
  "extends": ["@rcc-pkg/eslint-config"]
}
```
